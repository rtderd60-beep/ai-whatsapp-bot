# AI WhatsApp Bot

موقع بوت WhatsApp الذكي مع تحميل الفيديوهات

## الميزات:
- 🤖 ردود ذكية من AI
- 🎥 تحميل فيديوهات من جميع المنصات
- 📱 ربط سهل مع WhatsApp
- ⚡ سرعة فائقة

## البدء:
1. انسخ المستودع
2. شغّل `npm install`
3. أضف مفاتيح API الخاصة بك
4. شغّل `npm start`
